Hướng dẫn sử dụng
1.****Trên Windows: Mở Command Prompt hoặc PowerShell,
 di chuyển đến thư mục chứa tệp start-compose.bat
 và chạy lệnh start-compose.bat

2.****Trên Linux và Mac OS: Mở Terminal,
di chuyển đến thư mục chứa tệp start-compose.sh
và chạy lệnh chmod +x start-compose.sh để cấp quyền thực thi cho tệp,
sau đó chạy lệnh ./start-compose.sh
